PLACE YOUR SOUND FILE HERE
===========================

Place your sound files (.mp3, .wav, .ogg, etc.) in this directory. 
The extension will randomly select and play one of these files whenever a terminal error occurs.

The extension will play this sound whenever an error is detected in the terminal.

Supported formats:
- MP3 (recommended)
- WAV
- OGG

Note: If you use a different format, update the file extension in extension.js

Where to find sounds:
1. Record your own sounds
2. Use text-to-speech services to generate one
3. Download from free sound libraries (freesound.org, etc.)
4. Use an online audio converter if you have the sound in a different format
